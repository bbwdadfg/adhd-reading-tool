# 📥 安装指南

## 系统要求

- **浏览器**: Chrome 88+ 或基于 Chromium 的浏览器（Edge、Brave、Opera 等）
- **操作系统**: Windows 10+、macOS 10.14+、Linux（任意发行版）
- **内存**: 建议 4GB 以上（扩展本身占用极少）

## 安装步骤

### 方法一：Chrome Web Store（推荐）

1. **访问官网**
   - 打开 [adhdreading.org](https://adhdreading.org)
   - 点击"立即下载"按钮

2. **Chrome Web Store 安装**
   - 自动跳转到 Chrome Web Store
   - 点击"添加至 Chrome"
   - 在弹出窗口中确认"添加扩展程序"

3. **验证安装**
   - 浏览器右上角出现 ADHD Reading 图标
   - 点击图标，看到设置面板即安装成功

### 方法二：手动安装（开发者模式）

1. **下载扩展包**
   - 访问 [GitHub Releases](https://github.com/your-username/adhd-reading/releases)
   - 下载最新版本的 `.zip` 文件
   - 解压到本地文件夹

2. **启用开发者模式**
   - 打开 Chrome，访问 `chrome://extensions/`
   - 右上角开启"开发者模式"

3. **加载扩展**
   - 点击"加载已解压的扩展程序"
   - 选择解压后的文件夹
   - 扩展自动加载完成

## 首次设置

### 基础配置

1. **选择预设模式**
   ```
   轻度模式 - 适合偶尔注意力分散的用户
   中度模式 - 适合典型 ADHD 症状用户（推荐）
   重度模式 - 适合严重注意力缺陷用户
   自定义模式 - 完全个性化设置
   ```

2. **调整基础参数**
   - **整体放大**: 建议设置为 108%-115%
   - **文字间距**: 根据个人喜好，建议 8-12
   - **强调密度**: 推荐 35%-45%
   - **强调强度**: 建议 120%-130%

3. **选择强调模式**
   - **仿生阅读**: 单词前缀加粗（推荐）
   - **语义加粗**: AI 识别关键词加粗
   - **语义高亮**: AI 识别重要内容高亮

### 高级设置

1. **语言设置**
   - 自动检测网页语言
   - 手动指定主要阅读语言
   - 多语言混合内容优化

2. **网站管理**
   - 设置自动启用的网站
   - 添加排除网站列表
   - 敏感页面保护设置

3. **视觉定制**
   - 选择颜色主题
   - 调整高亮颜色
   - 设置段落缩进

## 使用技巧

### 快速操作

- **一键启用/禁用**: 点击扩展图标
- **快速切换模式**: 使用预设按钮
- **临时关闭**: 双击页面空白处

### 键盘快捷键

```
Ctrl + Shift + A  - 启用/禁用扩展
Ctrl + Shift + M  - 切换强调模式
Ctrl + Shift + R  - 重置当前页面
```

### 最佳实践

1. **渐进式调整**
   - 从中度模式开始
   - 每次只调整一个参数
   - 使用一周后再做微调

2. **场景化设置**
   - 学术阅读：使用重度模式
   - 新闻浏览：使用轻度模式
   - 技术文档：启用语义加粗

3. **定期优化**
   - 每月检查使用统计
   - 根据阅读效果调整参数
   - 尝试新功能和模式

## 常见问题

### 安装问题

**Q: 扩展安装后没有图标？**
A: 检查是否被隐藏在扩展菜单中，点击拼图图标查看所有扩展。

**Q: 提示"无法加载扩展程序"？**
A: 确保下载的是完整的扩展包，重新解压后再次加载。

**Q: Chrome Web Store 无法访问？**
A: 可以使用手动安装方式，或通过官网提供的镜像链接。

### 功能问题

**Q: 某些网站不生效？**
A: 检查是否在排除列表中，或者该网站使用了特殊的文本渲染方式。

**Q: 文字显示异常？**
A: 降低强调强度，或切换到其他强调模式。

**Q: 影响网页正常功能？**
A: 临时禁用扩展，或将该网站添加到排除列表。

### 性能问题

**Q: 页面加载变慢？**
A: 降低强调密度，或在设置中关闭实时分析功能。

**Q: 浏览器卡顿？**
A: 检查是否同时开启了过多扩展，建议关闭不必要的扩展。

## 卸载指南

### 完全卸载

1. **移除扩展**
   - 访问 `chrome://extensions/`
   - 找到 ADHD Reading
   - 点击"移除"按钮

2. **清理数据**
   - 扩展数据会自动清理
   - 如需手动清理，可重置浏览器设置

3. **反馈收集**
   - 卸载前欢迎提供反馈
   - 帮助我们改进产品体验
