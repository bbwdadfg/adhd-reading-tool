# 🚀 功能详细说明

## 核心功能概览

ADHD Reading 提供了一套完整的阅读辅助功能，专门针对注意力缺陷用户的需求进行优化。每个功能都经过科学验证和用户测试，确保能够真正改善阅读体验。

## 🔬 仿生阅读技术

### 工作原理

仿生阅读是我们的核心技术，基于神经科学研究中关于眼球运动和文字识别的发现。当我们阅读时，大脑并不是逐字处理，而是通过识别单词的前半部分来快速理解整个单词。

**科学依据**：
- 研究表明，熟练读者的眼球只在单词的前1/3处停留
- 大脑通过"预测性编码"来补全单词的后半部分
- 这种机制在 ADHD 用户中往往不够稳定

### 技术实现

```
原始文本: "The quick brown fox jumps over the lazy dog"
仿生处理: "**Th**e **qui**ck **bro**wn **f**ox **jum**ps **ov**er **th**e **la**zy **d**og"
```

**智能分割算法**：
- **英文单词**: 基于音节结构和词频分析
- **中文词汇**: 考虑语义权重和认知负荷
- **复合词**: 特殊处理连字符和缩写
- **数字和符号**: 保持原有格式不变

### 参数调节

1. **强调密度 (20%-60%)**
   - 控制被强调单词的比例
   - 低密度：适合轻度注意力问题
   - 高密度：适合严重注意力缺陷

2. **强调强度 (100%-160%)**
   - 控制加粗效果的明显程度
   - 100%：几乎不可见的轻微加粗
   - 160%：非常明显的重度加粗

3. **分割比例**
   - 自动根据单词长度调整
   - 短单词（1-3字符）：完全加粗
   - 中等单词（4-7字符）：前半部分
   - 长单词（8+字符）：前1/3部分

## 🤖 AI 语义分析

### 语义加粗模式

使用自然语言处理技术，智能识别文本中的关键信息并进行加粗处理。

**识别类型**：
- **核心概念**: 专业术语、重要名词
- **关键数据**: 数字、百分比、日期
- **动作词汇**: 重要的动词和动词短语
- **逻辑连接**: 因果关系、转折词汇

**算法特点**：
- 基于 TF-IDF 和语义相似度
- 考虑上下文语境
- 支持多语言混合文本
- 实时处理，无需预加载

### 语义高亮模式

更进一步的语义分析，不仅识别关键词，还能理解句子和段落的重要性。

**高亮策略**：
- **重要句子**: 包含核心信息的完整句子
- **关键段落**: 总结性或结论性段落
- **引用内容**: 重要的引用和数据
- **标题层次**: 自动识别标题重要性

**颜色编码**：
- 🔴 **红色**: 最重要的信息
- 🟡 **黄色**: 重要的支撑信息
- 🔵 **蓝色**: 补充说明信息
- 🟢 **绿色**: 示例和案例

## 📏 排版优化

### 文字调节

**整体放大 (90%-140%)**：
- 不仅仅是字体大小，还包括行高调整
- 保持文字比例协调
- 适配不同屏幕分辨率

**文字间距 (0-50)**：
- 字符间距（letter-spacing）调节
- 单词间距（word-spacing）优化
- 避免文字过于拥挤或分散

### 段落优化

**段首缩进**：
- 传统的段落缩进格式
- 帮助区分段落边界
- 可选择缩进大小

**行间距调节**：
- 自动计算最佳行间距
- 基于字体大小动态调整
- 减少行与行之间的视觉干扰

### 页面布局

**边距优化**：
- 增加页面两侧留白
- 减少单行文字数量
- 创造更舒适的阅读环境

**背景调节**：
- 可选择护眼背景色
- 降低屏幕蓝光影响
- 支持深色模式

## 🎯 聚焦辅助

### 段落高亮

**悬停高亮**：
- 鼠标悬停自动高亮当前段落
- 淡化其他内容，减少干扰
- 平滑的过渡动画效果

**点击锁定**：
- 点击段落可锁定高亮状态
- 适合需要长时间专注的内容
- 再次点击取消锁定

### 阅读进度

**进度指示**：
- 页面右侧显示阅读进度条
- 实时更新当前位置
- 点击可快速跳转

**阅读时间**：
- 估算剩余阅读时间
- 基于个人阅读速度调整
- 帮助合理安排时间
